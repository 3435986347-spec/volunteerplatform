# 雷州市恒德爱心公益协会志愿者管理平台 — 业务领域划分

> 基于《小程序设想【第十版】.xlsx》整理，覆盖小程序前端（首页/社区/我的）及后台管理全功能。

---

## 一、用户与身份认证

- 志愿者注册
  - 基础信息采集（姓名、身份证、手机号、政治面貌、学校、年级、通讯地址、i志愿者码）
  - 身份证二要素实名认证
  - 手机号短信验证码校验
  - 企业微信群成员前置校验（未入群则弹出群二维码）
  - 未成年人紧急联系方式收集
  - 志愿者协议阅读与手写签名
  - 会员费预留门槛（注册年限/年费）
- 爱心企业注册
  - 企业信息采集（企业名称、信用代码、企业介绍、负责人、电话验证码）
  - 账号/密码申请
  - 后台注册审核
- 登录与账号切换
  - 志愿者登录
  - 管理团队登录
  - 爱心企业登录
  - 游客状态（已登录未注册）
- 角色权限体系
  - 游客（仅浏览，受限操作）
  - 志愿者（实名认证后）
  - 活动临时负责人（通过考试的志愿者）
  - 管理团队（各部门子账号）
  - 爱心企业（独立账号）

---

## 二、志愿活动

- 活动发布
  - 活动照片、名称、时间段（精确到分钟）、地点、要求、内容
  - 时间段下设子项目，支持多时间段
  - 报名时间分层设置（管理团队 / 临时负责人 / 普通志愿者）
  - 活动积分设置
  - 报名限制（全平台 / 指定分队）
  - 弹窗公告（报名须知，可设置倒计时才可确认）
  - 报名成功提示（加群文字+图片）
  - 发布团队名称与联系人
- 活动报名
  - 志愿者自主报名（选择时间段）
  - 同小组成员代报名
  - 全平台同时间段活动防重复报名
- 活动签到/签退
  - 扫一扫（首页全局扫码入口，亦用于兑换核销、捐书扫码入库）
  - 二维码签到（ZXing 生成）
  - 活动负责人手动确认签到
  - 到达指定定位距离后自动签到
  - 活动结束后到家定位确认（详细地址仅最高权限可查）
- 活动名单公示
  - 固定模板展示报名名单
  - 普通志愿者电话中间打\*号，负责人全显示
  - 按活动时间段排列
  - 公示时间：名单确认后至活动开始，活动开始后自动撤销
  - 管理员可设置任意志愿者为活动负责人
- 活动评价
  - 志愿者对活动评价
  - 志愿者对活动负责人评价
  - 负责人对志愿者评价
- 活动相册
  - 以「活动编号+活动名称」自动建立相册
  - 记录上传人、上传数量
  - 默认上传原图，支持全图预览
  - 上传积分激励（审核后发放）
  - 分级上传/删除/查看权限（志愿者/负责人/管理团队/宣传部）
- 服务记录
  - 个人参与活动名称、签到签退时间、活动时长
  - 签到记录须经秘书部确认方可显示
  - 补录签到记录支持

## 三、积分体系

- 积分规则
  - 活动参与积分（完成活动后按规则发放）
  - 违规积分减半或不发放
  - 相册上传积分激励
- 积分中心（个人）
  - 总积分、已使用积分
  - 积分增减明细（含搜索框）
- 积分商品兑换
  - 商品浏览（图片、名称、所需积分、赞助单位）
  - 商品详情（评价、兑换记录、赞助企业主页入口）
  - 自提核销（条形码 + 随机码，ZXing 生成）
  - 快递寄送（支付快递费 / 积分抵扣，接入物流轨迹）
  - 兑换卷/满减卷（后台发放，支持批量）
  - 商品仅限指定卷才可兑换功能
  - 企业核销员扫码核销
  - 我的兑换记录、我的评价
  - 兑换规则说明（文字+图片）
- 积分流转（企业侧）
  - 志愿者用积分兑换后，企业积累该积分
  - 企业积分可兑换平台权益（如广告位）
- 排行榜
  - 活动次数排行、活动时长排行、积分排行、微心愿排行
  - 按月 / 年 / 总计分别排行
  - 历史月份下拉查询
- 勋章体系
  - 勋章样式管理、上传、审核、发放（均后台操作）
  - 个人主页展示勋章
- 奖惩中心
  - 各类违规记录
  - 各类奖励记录

## 四、公益捐赠

### 4.1 项目众筹

- 项目列表（全部 / 进行中 / 已结束）
- 项目详情（图片、名称、预计金额、进度、捐赠人数）
- 捐款方式：自定义金额 / 全款（微信支付）
- 捐物方式：填写物资名称、快递公司、快递单号（物流轨迹接入）
- 生成分享海报、邀请朋友
- 捐赠记录（捐款+捐物，物流轨迹可查）

### 4.2 公益捐书

- 活动信息展示（名称、时间、状态、活动数据、分享）
- 完整物资流转流程
  1. 志愿者报名（展示收件地址，含捐赠人姓名拼接）
  2. 捐赠人录入物资资料（物品条形码数据库比对核验）
  3. 生成包裹、录入快递单号及快递公司
  4. 机构扫码确认到货
  5. 机构扫码核对捐赠单据
  6. 为每件物品生成专属条形码（标签打印机，面单含物品名/受捐学生/受捐单位/孩子编号）
  7. 生成箱子条形码，扫码装箱
  8. 扫码确认箱子及物品到达受赠单位
- 不合格物资退回流程（后台添加/减少物品，生成退件快递单）
- 物资捐赠全轨迹对捐赠人公开
- 手机网页自适配（后台操作需用手机）
- 快递对接（快递100 API）

### 4.3 助学助困（结对）

- 项目类型：结对助学、结对助困、结对助残、结对成功
- 项目详情（图片、受助金额、已筹金额、参加人数）
- 捐赠完成后受助方可给捐赠人写信（图文展示板块）
- 自动生成捐赠证书
- 结对中心（个人捐赠记录，含开发票预留入口）

### 4.4 圆梦微心愿

- 心愿池（待认领）、已认领、已实现、榜单
- 心愿详情（编号、图片、心愿故事、基本信息，关键信息自动打\*）
- 认领后全资料展示给捐赠人
- 物资流转全流程（同捐书流程，对接快递100）
- 物资接收地址（快速复制）
- 捐赠人认领须为实名注册志愿者；查看心愿须登录并验证手机号

## 五、组织管理

- 组织架构
  - 树形方框结构展示（部门/职位/志愿者信息）
  - 框架新增/减少由最高权限操作
  - 后台插入志愿者后显示其部门与职位
- 志愿小组
  - 实名志愿者自由发起并命名小组
  - 发起人默认为小组负责人
  - 负责人可设管理员（≤3人）、移除成员、审批加入
  - 同小组成员可互相代报名活动
  - 一人只能加入一个小组
  - 搜索：名称 / 编号
  - 后台：搜索、批量导入、解散、组建、组长转移、变更记录
  - 新建小组需后台审核
- 归属组织（分队）
  - 分队类型由后台自定义
  - 后台设置每类型志愿者可加入数量上限
  - 未归属志愿者可查看分队负责人信息
  - 已归属志愿者可查看同分队名单
  - 后台设置分队内可见信息范围
  - 申请审核权归分队负责人，总账号可查全部申请记录
- 子账号与权限
  - 管理团队每人独立子账号
  - 细粒度功能权限，最高权限账号统一分配
  - 菜单权限、数据权限、审核流程权限分离
  - 界面水印（手机尾号 + 姓名 + 部门）
- 评优评先
  - 类问卷功能（单选、多选、判断、简答、文件上传、日期选择，预留）
- 报名管理团队
  - 类简历提交 / 问卷调查（单选、多选、判断、简答、文件上传、日期选择，预留）
- 活动临时负责人考试（资格与考试归组织管理领域）
  - 题型：单选、多选、判断、填空、简答
  - 达到分数线即获资格
  - 志愿者评价过低可由组织部取消资格

  - 企业介绍

  - 设置企业核销员

- 后台管理

  - 暂停、删除、批量导出
  - 代企业发布积分商品
  - 积分流转管理

## 七、社区交流

- 帖子流
  - 最热（当前1小时综合阅读/点赞/评论排序）
  - 关注（仅已关注用户帖子）
  - 官方（由各部门后台发布，前端显示分队/部门名称）
- 帖子功能
  - 发布（文字 / 图片 / 视频）
  - 查看、点赞、评论、分享、下载
  - 数据统计（阅读量/点赞量/评论量/分享量）
  - 帖子可见性设置（不限制/隐藏/仅关注/仅粉丝/禁评/禁赞）
  - 帖子修改
- 私信
  - 一对一私聊（参考抖音私信界面）
  - 聊天记录全量保存后台，仅最高管理员可查
  - 用户投诉私聊时审核员可查看聊天内容
- 互动通知
  - 他人评论/回复自己帖子的聚合提示
  - 点赞/评论订阅消息每20分钟汇总推送一次
- 用户主页
  - 头像、昵称、勋章、注册时间、发帖量、粉丝量、关注量、点赞量
  - 帖子列表、评论列表
  - 关注 / 私聊 / 备注 / @TA
- 个人主页隐私设置
  - 禁止关注、禁止评论、禁止点赞、不让TA看
- 内容安全
  - 关键词风控（后台配置，触发则自动插队审核）
  - 帖子审核（可设置审核次数与审核员）
  - 违规封号/禁言（禁发帖X天、禁评论X天、禁点赞X天）
  - 举报功能（长按评论 / 帖子）
  - 后台：管理、禁言、隐藏、删除、发布、置顶

## 十一、后台管理系统

- 首页看板
  - 平台核心数据概览
  - 小版块右下角快速跳转对应审核界面
- 用户管理
  - 志愿者全量信息查看、修改、暂停、删除、重置密码
  - 批量下载（含志愿者码和头像链接）
  - 搜索（姓名/手机号/身份证，下拉：性别/分队/政治面貌/学校/年级）
  - 自动识别身份证（年龄/性别/生日）
  - 职位信息维护（前端个人页面展示）
  - 爱心企业管理（查看/搜索/注册审核/暂停/删除/批量导出）
- 活动管理
  - 全平台活动列表及各项数据
  - 活动发布与管理（含子时间段）
  - 报名审核（不符合可直接删除，优先展示管理团队/临时负责人）
  - 手动新增志愿者报名，设置活动管理员
  - 各数据界面批量下载
- 积分与商品管理
  - 志愿者积分板块（总积分/已用积分/明细，搜索框）
  - 积分商品管理（审核/兑换管理，搜索：兑换人/姓名/电话/积分）
  - 商品隐藏、指定卷限制
  - 发放兑换卷/满减卷（批量发放）
  - 企业核销员设置
  - 积分商品代发布
- 分队管理
  - 设置/删除分队及负责人
  - 分队类型（学校分队/乡镇分队）
  - 志愿者注册时分队选择
- 服务记录管理
  - 全量签到/签退记录（含补录）
  - 秘书部确认后方可显示
  - 多维度筛选
- 捐赠管理
  - 众筹项目：上传/查看/管理/删除/批量下载/搜索/取消
  - 捐书管理：批量导出（含箱子码/条形码/捐赠人信息/快递信息/进度）
  - 手机端自适配（捐书操作）
  - 微心愿：批量上传/下载，取消认领，搜索/下拉筛选
  - 助学助困项目：批量上传/下载/管理
- 内容与社区管理
  - 帖子管理（发布/置顶/隐藏/删除/禁言）
  - 评价板块（活动评价/志愿者评价/负责人评价/商品评价/赞助商评价）
  - 聊天记录查看（仅最高管理员或被授权账号）
  - 审核功能（帖子/私聊内容）
